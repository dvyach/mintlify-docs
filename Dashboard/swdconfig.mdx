<Tabs>
  <Tab title="Software Distribution">
A support agent can only execute/ deploy a software packages that have already been configured. To do so:
1.	Navigate to the module via the side pane.
2.	Select the Site by clicking on (Change?)
3.	The table displayed on the page will display the following details:
    1.	Type of Operating system
    2.	Name of the package
    3.	Package description
    4.	Package version
    5.	Details regarding if the package has been configured for deploy/ execute.
4.	Select the Package that you want to execute/deploy and select 'Deploy & install' from the menu.
5.	Next, select appropriate answers for the questions 0 Do you want to Deploy? & Do you want to execute? If you have chosen No for both the questions, you will not be able to ’save’ the configuration.
6.	Click on OK to confirm that you want to perform the action that you have selected.

  <Tab title="Export Status of Software Distribution">
To export the status of Software Package distribution.
1.	Navigate to the Module and select the Site/ machine/ group and the site that you wish to export the status of.
2.	Click on the Menu on the far right and Select “Export status" from the drop down.
3.	The details would be exported as a .xls file and saved to your default download location.
