    ## MongoDB Data Modeling
    ### Unit Overview

 Follow these steps to monitor the Notification created by devices.
1. Navigate to the module via the side menu. 
2. Select the notification from the list on left hand side of the page.
3. Notification will contain the following details:
4. Name of the notification
5. Name of the device 
6. Date & time at which the notification was created
7. Status of the notification
8. Notes added, if any
